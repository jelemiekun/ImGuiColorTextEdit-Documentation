SelectionMode

Overview

Enum Class Name: TextEditor::SelectionMode

Represents the current mode of text selection in the editor. It defines how the selection behaves when the user clicks and drags with the mouse or uses keyboard commands. This affects whether selections are character-based, word-based, or line-based.

Enum Values

Type Name Description
Enum Line Selects an entire line.
Enum Normal Regular selection mode (character-wise).
Enum Word Selects a word at a time.
