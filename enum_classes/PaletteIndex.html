PaletteIndex

Overview

Enum Class Name: TextEditor::PaletteIndex

Represents the different color slots used by the editorâ€™s palette. Each syntax element (keywords, strings, comments, background, line number, etc.) uses a PaletteIndex to determine which color it should be rendered with.
The palette is essentially an array indexed by this enum.

This enum defines all highlightable categories in the editor.
