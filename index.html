Detailed Developer Guide

Class/Struct/Enum Class Description
TextEditor Main class representing the text editor. Handles text input, display, and editing.
TextEditor::Breakpoint Represents a breakpoint in the editor, typically for debugging or marking lines.
TextEditor::Coordinates Represents a position in the text (line and column).
TextEditor::Glyph Represents a single character/glyph, including its color and formatting.
TextEditor::Identifier Stores information about identifiers (like variables, functions) for syntax highlighting.
TextEditor::LanguageDefinition Defines syntax rules, keywords, and highlighting for a programming language.
TextEditor::EditorState Stores the current state of the editor (cursor position, selection, scroll, etc.).
TextEditor::UndoRecord Represents a single undo/redo action in the editor.
TextEditor::PaletteIndex Enum for different color roles in the editor (e.g., keyword, string, number).
TextEditor::SelectionMode Enum representing how text selection works (normal, word, line).
