TextEditor

Overview

Class Name: TextEditor

The TextEditor class is the main class in ImGuiColorTextEdit responsible for text editing functionalities. It provides a rich set of features such as syntax highlighting, cursor management, selection, undo/redo, and clipboard support. It is designed to be rendered inside an ImGui window.

            if (ImGui::Button("Paste")) editor.Paste();

            if (ImGui::Button("Select All")) editor.SelectAll();
