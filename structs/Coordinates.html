Coordinates

Overview

Struct Name: TextEditor::Coordinates

Represents a specific position in the editor text, defined by a line number and a column index. It is used for cursor positioning, selection, and text manipulation.

Constructor Summary

Constructor Description
Coordinates() Default constructor: initializes mLine and mColumn to 0.
Coordinates(int aLine, int aColumn) Parameterized constructor: initializes mLine and mColumn to provided values and asserts they are >= 0.
static Coordinates Invalid() Static function returning a Coordinates object with mLine and mColumn set to -1 (invalid coordinate).

Structs

Does not contain any nested structs or classes you can instantiate.

Members
