Breakpoint

Overview

Struct Name: TextEditor::Breakpoint

Represents a single breakpoint in the editor. Breakpoints are associated with specific lines of text and can be toggled on or off. They are commonly used in debugging scenarios to mark lines where execution should pause.

Constructor Summary

Constructor Description
Breakpoint() Default constructor; initializes mLine to -1, mEnabled to false, and mCondition to an empty string.

Structs

Does not contain any nested structs or classes you can instantiate.

Functions

Does not contain any function members you can call.

Members

Type Name Description
int mLine The line number where the breakpoint is set; initialized to -1 by default.
bool mEnabled Determines if the breakpoint is active; initialized to false by default.
std::string mCondition Optional condition for the breakpoint; empty by default.

        #include "imgui.h"

        TextEditor editor;

            editor.SetText("

            TextEditor::Breakpoint bp1;

            editor.SetBreakpoints(breakpoints);
        }
