Identifier

Overview

Struct Name: TextEditor::Identifier

Represents the type of a piece of text (token) for syntax highlighting. Each glyph or token in the editor can be assigned an Identifier to indicate its role (e.g., keyword, number, string, comment). This allows TextEditor to colorize text appropriately.
