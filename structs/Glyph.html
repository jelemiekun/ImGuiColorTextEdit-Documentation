Glyph

Overview

Struct Name: TextEditor::Glyph

Represents a single character (glyph) in the text editor, along with its visual and semantic attributes, such as color, identifier type, and whether it is part of a selection. Glyphs are used internally to render the editor’s text with syntax highlighting.

Constructor Summary

Constructor Description
Glyph(Char aChar, PaletteIndex aColorIndex) Parameterized constructor: initializes the character (mChar) and its color index (mColorIndex). Boolean flags mComment, mMultiLineComment, and mPreprocessor are initialized to false.

Members

Type Name Description
Char mChar The character stored in the glyph.
PaletteIndex mColorIndex The color of the glyph, represented by a PaletteIndex enum value.
bool mComment Flag indicating if the glyph is part of a comment (single-line).
bool mMultiLineComment Flag indicating if the glyph is part of a multi-line comment.
bool mPreprocessor Flag indicating if the glyph is part of a preprocessor directive.
