Class Name: TextEditor::UndoRecord

Represents a single undo/redo action in the editorâ€™s history. An UndoRecord captures all text changes performed during an edit operation, including inserted or removed characters, as well as cursor/selection states before and after the change.
This struct makes it possible for the editor to accurately undo or redo modifications while preserving the exact cursor and selection positions.
